## 1. 系统简介

行政办公任务中心系统是一个用于管理、分配和跟踪行政办公任务的综合平台。系统支持任务创建、分配、跟踪、协作和统计分析等功能，旨在提高团队协作效率和任务管理透明度。

### 1.1 主要功能

- 任务创建与管理
- 多级子任务支持
- 任务分配与状态跟踪
- 协作评论与附件管理
- 通知提醒
- 任务进度统计与报表

### 1.2 用户角色

系统定义了以下几种主要用户角色：

- **普通用户**：可以查看和处理分配给自己的任务
- **部门经理**：可以管理部门内的任务和人员
- **任务管理员**：可以创建和管理全局任务
- **系统管理员**：拥有系统的完整管理权限
