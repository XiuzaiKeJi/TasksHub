## 1. 技术栈概述

本系统前端基于以下技术栈开发：

- **框架**：React 18.x
- **类型检查**：TypeScript 5.x
- **状态管理**：Redux Toolkit
- **路由**：React Router 6.x
- **UI组件库**：Ant Design 5.x
- **样式处理**：Less + CSS Modules
- **网络请求**：Axios
- **构建工具**：Vite
